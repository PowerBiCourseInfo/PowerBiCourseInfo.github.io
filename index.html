Курс

POWER BI це:
• аналіз даних з найрізноманітніших джерел;
• візуалізація даних;
• найсучасніша система звітності та аналітики.

Power BI

Курс «Аналіз даних з MS Power BI Desktop» призначений для формування у користувача практичних навичок використання програми MS Power BI Desktop як інструменту для бізнес-аналітики. Студенти отримують необхідні методичні рекомендації щодо виконання практичних робіт у вигляді текстових та відеофайлів. На заняттях розглядаються різноманітні приклади, а виконання домашніх завдань дозволить закріпити отримані навички в позаурочний час

ДИПЛОМ
Наприкінці курсу, після успішного виконання домашніх завдань та тестування, видається диплом.

Далі

ПРОГРАМА КУРСУ

✓


• Загальний огляд сімейства програм MS POWER BI: Power BI Desktop, Power BI Service, Power BI Mobile, Power Pivot, Power Query
• Завантаження даних з файлів формату .csv, .txt та PDF.
• Завантаження даних з файлів Excel. Масове завантаження даних з папки.
• Очищення, модифікація та трансформація даних у вікні редактора Power Query.
• Завантаження даних з інтернету.
• Створення моделі даних.

✓

• Групування, впорядкування та фільтрація даних.
• Аналіз даних в Power BI. Перегляд статистичних даних. Виявлення викидів за допомогою візуальних елементів Power BI. Аналіз часових рядів. Використання функції "Аналіз". Пошук важливих факторів за допомогою візуального елемента "Основні чинники впливу”. Використання візуального елемента «Структурне дерево" для розбиття міри. Створення та використання параметрів What if для візуалізації змінних у Power BI Desktop
• Функції DAX для роботи з датами. Створення таблиць дат
• Створення звіту, його форматування та налаштування
• Публікація звіту

✓

• Завантаження інформації з баз даних
• Злиття запитів додаванням та об’єднанням
• Загальні відомості про функціональну мову DAX. Базові функції DAX: статистичні функції, логічні
функції, функції фільтрації тощо
• Створення обчислювальних стовпців і таблиць
• Візуальні елементи MS Power BI. Особливості створення візуалізацій різних типів
• Крос-фільтрація та перехресне виділення

Що ви отримуєте

Після закінчення курсу Ви будете знатися на:
• завантаженні даних з різних джерел;
• підготовці даних для подальшого аналізу і візуалізації;
• створенні запитів різного ступеню складності;
• обробці та аналізі отриманих даних;
• побудові моделі даних;
• використанні функціональної мови DAX;
• створенні та застосуванні фільтрів;
• створенні та налаштуванні звітів різного ступеню складності.
